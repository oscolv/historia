## Estado de México antes de 1869

Este shapefile representa el territorio del Estado de México antes de la creación de los estados de Hidalgo y Morelos.

Fue generado mediante la unión de los límites actuales de:
- Estado de México
- Hidalgo
- Morelos
- Calpulalpan (Tlaxcala)

Proyección: EPSG:6372 (Mexico ITRF2008 / LCC)

Área total: 48,306.043 km²

Reconstrucción cartográfica, 2026.
